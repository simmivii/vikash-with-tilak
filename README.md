# Vikash Simple Wedding Website

A beautiful and elegant wedding invitation website featuring a countdown timer, event details, and responsive design.

## Features

- **Hero Section**: Elegant gradient header with wedding announcement
- **Live Countdown Timer**: Real-time countdown to the wedding date (April 25, 2026)
- **Event Details**: Information about wedding ceremonies and celebrations
- **Responsive Design**: Optimized for desktop and mobile devices
- **Map Integration**: Direct links to venue locations
- **Beautiful Styling**: Soft pink and rose color scheme perfect for weddings

## Files Structure

```
vikash-simpel-wedding/
├── index.html          # Main HTML file with website structure
├── style.css           # CSS styles for beautiful presentation
├── script.js           # JavaScript for countdown timer functionality
├── images/             # Directory for wedding photos and images
└── README.md           # This documentation file
```

## Setup Instructions

1. **Clone or Download** this project to your local machine
2. **Open** `index.html` in your web browser
3. **Customize** the content:
   - Update wedding date in `script.js` (line 1)
   - Replace placeholder text in `index.html`
   - Add your wedding photos to the `images/` folder
   - Modify colors and styling in `style.css` if desired

## Customization

### Changing the Wedding Date
Edit the date in `script.js`:
```javascript
const weddingDate = new Date("April 25, 2026 18:00:00").getTime();
```

### Adding Images
Place your wedding photos in the `images/` folder and update the image sources in `index.html`.

### Modifying Colors
The main color scheme is defined in `style.css`:
- Primary pink: `#b76e79`
- Light pink: `#f3c6d4`
- Accent pink: `#a13a5d`
- Background: `#fff1f6`
